# Инструкция по загрузке проекта на ваш GitHub

Так как у меня нет почты и своего аккаунта GitHub (я ИИ-ассистент, запущенный на вашем сервере), я не могу "согласиться на инвайт". 
Зато я имею доступ к вашему локальному серверу (`/root/.openclaw/workspace/`), где лежат все файлы проекта. 

Чтобы отправить файлы на ваш GitHub из командной строки вашего сервера, нужно сделать следующее:

## 1. Создайте пустой репозиторий на GitHub
1. Зайдите на GitHub под своим аккаунтом.
2. В верхнем правом углу нажмите `+` -> `New repository`.
3. Назовите его `crypto-agency-dashboard` (или как вам нравится).
4. **Важно:** Не ставьте галочки "Add a README file" или "Add .gitignore". Репозиторий должен быть абсолютно пустым.
5. Нажмите `Create repository`.
6. Вы увидите страницу с инструкциями. Вам нужна HTTPS ссылка на репозиторий, она выглядит примерно так:
   `https://github.com/ВАШ_ЛОГИН/crypto-agency-dashboard.git`

## 2. Создайте токен доступа (Personal Access Token)
Так как пароли больше не работают для git push, нужен токен:
1. В правом верхнем углу GitHub нажмите на свое фото (Settings).
2. Слева в самом низу выберите `Developer settings` -> `Personal access tokens` -> `Tokens (classic)`.
3. Нажмите `Generate new token` -> `Generate new token (classic)`.
4. Назовите его "OpenClaw Deploy", срок действия можно поставить 30 дней.
5. Отметьте галочку `repo` (Full control of private repositories).
6. Нажмите `Generate token` в самом низу.
7. **Скопируйте этот токен.** Он выглядит как `ghp_...` и показывается только один раз.

## 3. Выполните загрузку с вашего сервера
Теперь вернитесь в консоль вашего сервера (где вы устанавливали python-venv) и выполните эти команды, подставив ваши данные:

```bash
cd /root/.openclaw/workspace/dashboard/

# Инициализируем git
git init
git add .
git commit -m "Initial commit: AI Telegram Agency Dashboard"

# Добавляем ссылку на ваш репозиторий с токеном внутри для авторизации
# ВАЖНО: замените ВАШ_ТОКЕН, ВАШ_ЛОГИН и ИМЯ_РЕПОЗИТОРИЯ на свои!
git branch -M main
git remote add origin https://ВАШ_ТОКЕН@github.com/ВАШ_ЛОГИН/ИМЯ_РЕПОЗИТОРИЯ.git

# Отправляем файлы в GitHub
git push -u origin main
```

После этого весь код, который я написал, окажется в вашем репозитории на GitHub!
